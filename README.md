# Aplicaci-ndemoGrupo-3
 Características PrincipalesAutenticación: 
 Control de acceso mediante el nuevo flujo @if de Angular 18.Operaciones CRUD: Registro, lectura, actualización y borrado de productos.Interfaz Dinámica: Títulos y botones que cambian de        estado según la acción (Crear/Editar) mediante operadores ternarios.
 
 Reactividad: Uso de Two-Way Data Binding ([(ngModel)]) para la captura de datos en tiempo real.
 
 Rendimiento: Implementación de @for con la cláusula track para un renderizado eficiente de listas.
 
 🛠️ Tecnologías UtilizadasFramework: Angular 18 (Standalone Components).
 
 Lenguaje: TypeScript.
 
 Estilos: CSS nativo (Flexbox para diseño responsivo).
 
 📋 Requisitos e InstalaciónPara ejecutar este proyecto localmente, asegúrate de tener instalado Node.js y npm.
 
 Clonar el repositorio: https://github.com/RoberthLagos/AplicaciondemoGrupo-3.git
